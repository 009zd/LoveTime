# 使用方法
1. 第一步 从github 上下载

   `git clone https://github.com/songluyi/LoveTime.git`

2. 安装依赖库

    `pip install -r requirements.txt`

3. 将QQ 聊天记录导出并放置到msg 文件夹下面

    **如何导出？**

    导出QQ消息记录：http://jingyan.baidu.com/article/a501d80c33919fec630f5e82.html

    **只能放置一个么**

    是的

4. 运行moniter.py

`   python moniter.py`
