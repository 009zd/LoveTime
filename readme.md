## 说明
将QQ聊天记录（txt文件）放置到msg后运行moniter.py就可以直观的看到聊天的分析现在由于还未完成，如果你现在就像看到，那么可以这样先运行get2db然后运行moniter就好
1.	聊天对象的信息（爬虫做的）
2.	第一次聊天的时间，历史上聊天频次分布表，每天聊天频次分布表，平均消息回复速度，消息回复速率增减表。
3.	可视化你们的聊天轨迹哈哈哈哈哈哈

安装依赖包：matplotlib, wordcloud, colorama
